URL NECESARIAS 

📦 tbl_admin
Método	URL	Descripción
GET	http://localhost:3000/admin	Obtener todos los admin
GET	http://localhost:3000/admin/:id	Obtener admin por ID
POST	http://localhost:3000/admin	Crear nuevo admin
PUT	http://localhost:3000/admin/:id	Actualizar admin por ID
DELETE	http://localhost:3000/admin/:id	Eliminar admin por ID

PARA PODER ENTRAR 

para inicializar el node.js y obtener las url y realizar consultas debemos inicializarlo en una terminal de visual studio 
1.cd backend
node server.js

inicializar en una terminal en visual, en veu podemos interactuar con la interfaz grafica 
2.cd frontend
npm run dev
